## Installation

### Install
```
$ git clone git@github.com:yaoqs/hexo-theme-scholar-future.git themes/scholar-future
```
### Enable

Modify theme setting in _config.yml to scholar-future.

### Update
```
cd themes/scholar-future
git pull
```
